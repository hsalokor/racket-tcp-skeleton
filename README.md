# Dependencies

* Racket 5.3.x

# Building

The files can be loaded to REPL, or compiled to executables using

    make

# Running

Compiled versions can be 

    ./server

or

    ./client

# Status

Working:

- Socket listening
- Socked connecting
- Simple JSON writing

Missing/broken/unknown:

- Socket writing or reading (does not read line properly)
- JSON reading
